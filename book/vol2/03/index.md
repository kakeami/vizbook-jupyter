# 3章 内訳を見る手法

質的変数の内訳を見るための可視化手法を学びます。水準間の割合や関係性を理解するための手法を扱います。

| セクション | 概要 | 主な関数 |
|-----------|------|----------|
| [円グラフ](./pie) | 各カテゴリの割合を扇形で表現 | `px.pie` |
| [棒グラフ](./bar) | 内訳の比較を棒の長さで表現 | `px.bar` |
| [モザイクプロット](./mosaic) | 複数変数の組み合わせ割合を表現 | `px.treemap` |
| [積上げ密度プロット](./area) | 連続的な内訳変化を面積で表現 | `px.area` |
| [ツリーマップ](./tree) | 階層構造の内訳を入れ子状に表現 | `px.treemap` |
| [パラレルセットグラフ](./parallel) | 複数変数の関係をリボンで表現 | `px.parallel_categories` |
