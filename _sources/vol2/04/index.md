# 4章 関係を見る手法

変数間の関係を見るための可視化手法を学びます。二変数の関係から時系列変化まで、様々な関係性を表現します。

| セクション | 概要 | 主な関数 |
|-----------|------|----------|
| [散布図](./scatter) | 二つの量的変数の関係を点で表現 | `px.scatter` |
| [バブルチャート](./bubble) | 三つの量的変数の関係をサイズで拡張 | `px.scatter` (size) |
| [散布図行列](./matrix) | 複数変数の組み合わせを一覧表示 | `px.scatter_matrix` |
| [二次元ヒストグラム](./2d) | 密集点の分布を色で表現 | `px.density_heatmap` |
| [等値線図](./contour) | 分布形状を等高線で表現 | `px.density_contour` |
| [折れ線グラフ](./line) | 時系列変化を線で表現 | `px.line` |
| [連結散布図](./connected) | 時系列の二変数変化を軌跡で表現 | `px.line` + marker |
