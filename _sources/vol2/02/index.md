# 2章 分布を見る手法

量的変数の分布を見るための可視化手法を学びます。データの全体的な傾向や散らばりを理解するための手法を扱います。

| セクション | 概要 | 主な関数 |
|-----------|------|----------|
| [ヒストグラム](./hist) | データをビンに分割し度数を棒で表現 | `px.histogram` |
| [密度プロット](./density) | 滑らかな曲線で分布形状を表現 | `ff.create_distplot` |
| [箱ひげ図](./box) | 分布の要約統計量を視覚化 | `px.box` |
| [バイオリンプロット](./violin) | 分布形状と要約統計量を同時に表現 | `px.violin` |
| [リッジラインプロット](./ridge) | 複数分布を重ねて変化を表現 | `px.violin` |
