# 6章 メディア展開データ分析②

メディア展開データを用いて、これまで学んだ可視化手法を総合的に実践します。量・分布・内訳・関係の観点から分析します。

| セクション | 概要 | 主な可視化 |
|-----------|------|------------|
| [メディア展開データの量を見る](./amounts) | アニメ化作品数や放送話数の比較 | 棒グラフ、ヒートマップ |
| [メディア展開データの分布を見る](./dists) | 連載期間や掲載位置の分布分析 | ヒストグラム、箱ひげ図 |
| [メディア展開データの内訳を見る](./props) | アニメ化率や雑誌別割合の把握 | 円グラフ、積上げ棒グラフ |
| [メディア展開データの関係を見る](./assocs) | 掲載特性とアニメ化の関係 | 散布図、等値線図 |
