# 練習問題一覧

本書の全74問の練習問題を章ごとに一覧で確認できます。

## 難易度の目安

| 難易度 | 説明 |
|:------:|------|
| {bdg-info}`基礎` | 本文とほぼ同じ処理（パラメータ変更程度） |
| {bdg-success}`標準` | 軽い前処理を含む（データ絞り込み等） |
| {bdg-warning}`発展` | 複数ステップの加工、応用的な分析 |
| {bdg-danger}`応用` | 複雑なデータ結合、高度な可視化技法 |
